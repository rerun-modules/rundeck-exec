COMMANDS

  • job: execute the job
  • output-text: print execution output to stdout
  • status: check a rundeck execution status

OPTIONS

  • api-token: the API access token
  • args: args to pass to the job
  • detail: show more detail
  • filter: node filter
  • format: the text format (text|plain|json)
  • id: the unique identifier
  • last-lines: only show specified number of lines
  • nodename: the nodename to use
  • server-url: rundeck base url
